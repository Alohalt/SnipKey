macOS 13+ · Swift 5.9+ · 开源

SnipKey

macOS 菜单栏文本扩展工具。在任意应用中输入 #trigger， 通过补全面板选择或直接展开为预设文本。

查看 GitHub 仓库 快速开始 下载 Release

核心能力

全局触发

系统范围内监听键盘输入，捕获 # 前缀触发词。

跟随光标的补全面板

浮动面板展示匹配结果，支持上下选择，Tab / Enter 确认，Esc 取消。

菜单栏常驻

快速打开权限引导、设置窗口和剪贴板记录，不打扰主界面。

原生三栏设置

分组、搜索、创建、编辑、删除，以及 JSON 导入导出。

动态变量

在替换文本中嵌入运行时上下文：

  • {date}
  • {time}
  • {clipboard}
  • {cursor}

剪贴板智能建议

记录剪贴板历史，重复复制时建议一键生成新的 Key，本地生成简短 trigger。

使用频率排序

按接受次数排序候选，常用 Key 优先展示。

触发词校验

仅允许字母、数字、下划线，保存时执行唯一性校验，导入时自动归一化。

快速开始

推荐使用已签名的开发包运行，让 macOS 把辅助功能权限绑定到稳定的应用身份上：

make signing-help
make bootstrap-personal-team
make run

本机已经有可用 Apple Development 证书时，通常这样就够了：

make test
make run

更完整的签名说明见 docs/development-signing.md。

使用方式

  1. 启动应用后，从菜单栏打开「设置」。
  2. 新建一个 Key，例如触发词 account，替换内容填入需要插入的文本。
  3. 回到任意可输入文本的应用，输入 #account。
  4. 使用 Tab、Enter 或终止字符确认展开。

展开时，SnipKey 会删除已输入的触发内容，再插入解析后的结果文本。

数据存储

  • Key 数据：~/Library/Application Support/SnipKey/snippets.json
  • 剪贴板历史：~/Library/Application Support/SnipKey/clipboard-history.json（界面滚动保留最近 50 条）
